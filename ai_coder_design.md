# AI Coder - 移动端AI编程软件设计文档

## 产品概述

AI Coder 是一款基于Flutter开发的移动端AI编程IDE，通过SSH连接到远程电脑或虚拟机，让用户可以在手机上进行完整的编程工作。核心特色是集成了Vibe Coding AI对话系统，让AI Agent可以直接修改代码、运行命令、管理Git等。

---

## 核心功能架构

### 1. 连接管理层

#### 1.1 SSH连接管理
- **连接配置**
  - 保存多个SSH连接配置（主机地址、端口、用户名、认证方式）
  - 支持密码认证和SSH密钥认证（RSA/ED25519）
  - 密钥管理（导入、生成、密码保护）
  - 连接分组和标签管理

- **连接状态**
  - 实时连接状态监控
  - 自动重连机制
  - 连接质量检测（延迟、丢包率）
  - 多连接同时管理

- **安全特性**
  - 本地加密存储凭据
  - 支持SSH Agent转发
  - 连接超时和心跳保活

#### 1.2 SMB文件共享（可选扩展）
- SMB/CIFS协议支持
- 直接挂载远程文件夹
- 与SSH连接协同工作

## 技术架构

### 技术栈

- **前端框架**: Flutter
- **状态管理**: Riverpod / BLoC
- **本地存储**: Hive / SQLite
- **SSH客户端**: dartssh2 或原生插件
- **终端模拟**: xterm.dart 或自定义实现
- **代码编辑**: flutter_code_editor
- **语法高亮**: highlight.dart
- **Git操作**: 通过SSH执行git命令
- **AI API**: HTTP客户端 + SSE支持

### 数据流架构

```
┌─────────────┐     ┌─────────────┐     ┌─────────────┐
│   UI层      │────▶│  状态管理层  │────▶│  服务层     │
│  (Flutter)  │◀────│ (Riverpod)  │◀────│ (Services)  │
└─────────────┘     └─────────────┘     └──────┬──────┘
                                               │
                          ┌────────────────────┼────────────────────┐
                          ▼                    ▼                    ▼
                   ┌─────────────┐      ┌─────────────┐      ┌─────────────┐
                   │ SSH服务     │      │ AI服务      │      │ 文件服务    │
                   │ (dartssh2)  │      │ (HTTP/SSE)  │      │ (SFTP/SCP)  │
                   └─────────────┘      └─────────────┘      └─────────────┘
```

## 核心交互流程

### 1. 初始设置流程
```
启动App → 欢迎页 → 添加SSH连接 → 测试连接 → 创建/选择项目 → 进入工作区
```

### 2. Vibe Coding工作流程
```
1. 用户在AI对话框输入需求
2. AI分析需求并规划操作步骤
3. AI通过SSH执行文件读取/命令执行
4. AI生成代码变更建议
5. 用户预览Diff并确认
6. AI应用变更到远程文件
7. 可选：AI执行编译/测试验证
```

### 3. Git工作流程
```
1. 用户修改代码（手动或AI辅助）
2. 查看Git状态面板
3. 选择要提交的文件
4. AI生成或用户输入commit message
5. 执行提交
6. 推送到远程仓库
```

## 总结

AI Coder 的核心价值在于：
1. **移动优先** - 专为手机设计的编程体验
2. **AI原生** - 从设计之初就集成AI能力，不是简单的附加功能
3. **远程开发** - 利用现有计算资源，手机只是交互终端
4. **Vibe Coding** - 让编程变得像对话一样自然

这个设计文档涵盖了产品的核心功能和技术架构，可以作为开发的基础蓝图。
